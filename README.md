作者は、画像の機能を実装するのが、苦手だったので、
今回、動くソースコードを参考に試行錯誤して、
サンプルアプリを作成した。

1. StatefulWidgeでアプリを作成して仕組みを理解する.
2. Riverpodにリファクタリングする.

使いたい機能を作れるようになるのが目的。
